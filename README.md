# NanaAppBACKEND

## Project Overview
Nana is a child-friendly mobile app to help parents, caregivers, and doctors record and track children's pain.

### Features
- User authentication (parent/child/doctor)
- Pain entry recording (location, intensity, type, time)
- Notes/comments for context
- Multi-child support
- Doctor access to review records

### Tech Stack
- Node.js
- Express.js
- MongoDB
- REST API
- JWT Authentication

### Branches
- main (production-ready)
- develop (active development)
- staging (testing/integration)
- feature/<name> (new features)
- fix/<name> (bug fixes)

### Setup
1. Clone repo
2. Install dependencies: `npm install`
3. Configure `.env` with PORT, DB_URI, JWT_SECRET
4. Start server: `npm start`
